# Example Integration

## Example prompt bundle

- Load `templates/SOUL.md`
- Load `templates/SYSTEM_PROMPT.md`
- Keep `templates/SPA_PROTOCOL.md` available for long sessions
- Write/reset using `templates/STATE_HANDOFF.json`

## Example behavior

### Normal mode
"Help me organize the next three priorities for my project."

Expected response style:
- concise summary
- prioritized next steps
- calm tone
- optional light wit

### Witness mode
"Witness this transcript and tell me only what can be verified."

Expected response style:
- Source
- Observed
- Cannot confirm
- Unobserved
